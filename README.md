# Form Design & Validation

A simple registration form created using HTML, CSS, and JavaScript. The project validates user input before allowing form submission.

## Features
- Name validation
- Email validation
- Password validation
- Error messages for invalid inputs
- Success message on valid submission
- Responsive design

## Technologies Used
- HTML
- CSS
- JavaScript

## Files
- `index.html` – Structure of the form
- `style.css` – Styling of the webpage
- `script.js` – Form validation logic

## How to Run
1. Download or clone the repository.
2. Open `index.html` in your web browser.
3. Fill in the form and test the validation.
